Indice de terminos

Default DNS Diodo Digitalizar Directorio Download DVD

Default

Configuración predeterminada de un sistema, programa o dispositivo, que puede ser modificada según las necesidades del usuario.

DNS

Sistema que traduce nombres de dominio (como www.google.com) en direcciones IP numéricas, permitiendo la conexión a sitios web.

Diodo

Componente electrónico que permite el paso de corriente eléctrica en una sola dirección, utilizado en circuitos electrónicos y fuentes de alimentación.

Digitalizar

Proceso de convertir información analógica (como documentos o imágenes en papel) en formato digital, utilizando escáneres o cámaras.

Directorio

Estructura de almacenamiento en un sistema operativo que organiza archivos y carpetas dentro de un disco duro o unidad de almacenamiento.

Download

Acción de transferir archivos desde Internet o un servidor a una computadora o dispositivo local.

DVD

Disco óptico utilizado para almacenar grandes cantidades de datos, como películas, software y archivos multimedia
