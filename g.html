Gateway Gigabyte

Gateway

Dispositivo o software que actúa como intermediario entre redes diferentes, permitiendo la comunicación entre ellas. Es común en redes domésticas para conectar dispositivos a Internet.

Gigabyte

Unidad de medida de almacenamiento digital equivalente a aproximadamente mil millones de bytes (1 GB = 1,024 megabytes). Se utiliza para medir la capacidad de discos duros, memorias USB, archivos, etc.
