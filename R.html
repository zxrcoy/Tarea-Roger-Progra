Indice de terminos

RAM (Random Access Memory) Realidad virtual ROM (Read-Only Memory) Router Red

RAM (Random Access Memory)

Memoria de acceso rápido utilizada para almacenar temporalmente datos y programas en ejecución.

Realidad virtual

: Tecnología que crea entornos digitales simulados en los que los usuarios pueden interactuar a través de dispositivos como gafas VR.

ROM (Read-Only Memory)

Memoria de solo lectura que almacena datos permanentes en dispositivos electrónicos.

Router

Dispositivo que dirige el tráfico de datos entre redes y permite la conexión a Internet.

Red

Conjunto de dispositivos interconectados que comparten información y recursos, como Internet o redes locales (LAN).
