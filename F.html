FAQ

Sección común en sitios web o manuales que contiene respuestas a preguntas frecuentes hechas por los usuarios. Sirve para resolver dudas rápidamente sin necesidad de contacto directo.

Fatal error

Tipo de error crítico que provoca que un programa se detenga o que el sistema operativo se bloquee. A menudo requiere reiniciar el sistema o cerrar la aplicación afectada.

Fibra óptica

Medio de transmisión de datos que utiliza hilos de vidrio o plástico y señales de luz. Permite velocidades de conexión mucho más altas que los cables de cobre tradicionales y es muy utilizado en servicios de Internet modernos.

File

Unidad de almacenamiento digital que contiene información, como documentos, imágenes, música o programas. Todos los archivos tienen un nombre y una extensión (como .docx, .jpg, .exe).

Firewall

Sistema de seguridad que monitorea y controla el tráfico de red para permitir o bloquear accesos según un conjunto de reglas. Protege a los sistemas de accesos no autorizados o maliciosos.

Flag

Indicador o marcador binario que señala un estado específico dentro de un programa o sistema operativo. Por ejemplo, un flag puede indicar si una operación fue completada correctamente o si un bit está activado.
