Indice de terminos

Icono Inteligencia artificial Internet Intranet IP

Icono

Representación visual (normalmente pequeña) que simboliza una aplicación, archivo o función en una interfaz gráfica. Facilita la interacción del usuario con el sistema operativo o software.

Inteligencia artificial

Rama de la informática que crea sistemas capaces de realizar tareas que normalmente requieren inteligencia humana, como el reconocimiento de voz, el aprendizaje, la toma de decisiones y la traducción automática.

Internet

Red global de computadoras interconectadas que permite compartir información y servicios a través de protocolos como HTTP, FTP y otros. Es la base de servicios como el correo electrónico, las páginas web, el streaming, las redes sociales y más.

Intranet

Red privada utilizada dentro de una organización o empresa, similar a Internet, pero con acceso restringido solo a sus miembros. Se usa para compartir documentos, calendarios, aplicaciones internas y recursos de trabajo.

IP

Protocolo de comunicación que identifica de forma única a cada dispositivo en una red mediante una dirección IP (como 192.168.1.1). Es esencial para que los datos lleguen correctamente a su destino en Internet.
